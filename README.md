# MERN ToDo App

A simple **ToDo application** built with the **MERN stack** (MongoDB, Express, React, Node.js) to help users manage their tasks efficiently.

## Features

- Add, edit, and delete tasks
- Responsive UI with React
- RESTful API with Express and Node.js
- MongoDB database for persistent storage

## Tech Stack

- **Frontend:** React, CSS/Bootstrap
- **Backend:** Node.js, Express.js
- **Database:** MongoDB / Mongoose
- **Other:** Axios (for API calls), JWT (for Authentication)

## Installation

### Clone the repository
```bash
git clone https://github.com/Ravikumar-Rajendhiran/mern-todo-app.git
cd mern-todo-app

## Backend Setup

cd backend
npm install
npm start

## Frontend Setup

cd backend
npm install
npm start

## Usage

1.Open the app in your browser.
2.Add a new task using the input field.
3.Edit or delete tasks as needed.
